Installing Django (Web Framework) in Python

Posted 09 January 2020

Django is a high-level Python Web framework that encourages rapid development and clean, pragmatic design. Built by experienced developers, it takes care of much of the hassle of Web development, so you can focus on writing your app without needing to reinvent the wheel. It’s free and open source.

Django takes security seriously and helps developers avoid many common security mistakes. Some of the busiest sites on the Web leverage Django’s ability to quickly and flexibly scale.

To install, you need to make sure you have python Installed on your computer. To install python refer to www.python.org.

Django can be installed easily using pip on a system where you have python installed on. In the command prompt or terminal, you can execute the following command: py -m pip install Django or pip install django. This will download and install the latest Django release. After the installation has completed, you can verify your Django installation by executing Django-admin --version in the command prompt.

  • Django is compatible with major operating systems and databases.
  • It provides robust security features for your web application..
  • All components in your applications get deployed together..

The backside of the framework is that it relies heavily on the ORM system and makes web application components tightly coupled.

Cloud Computing - Big Picture

Posted 13 January 2020

Cloud computing can be categorized into three categories. These are cloud platforms, cloud applications, and private clouds.

The cloud application is generally used for an application that is running in a cloud environment, which is commonly referred to as Software as a service (SaaS). .

These applications are multi-tenancy which means customers are using one single application, but each customer has its own data which implies that those cloud applications are on a compute network own by some vendors, e-g Salesforce CRM.

Cloud Platforms provide services like virtual machines and storage and so on through remote provider access across the internet. These platforms are suitable for developers.

One key benefit is to pay for what you use and no more. The computer technology of the Cloud platform is IaaS (Infrastructure as a service). Some vendors that offer Cloud platforms are amazon web services, Microsoft Azure and others.

And the last cloud computing is private clouds. This is when an organization have their application running on-premises server but there is still a need for cloud application (SaaS) and cloud platforms (Iaas) to provide application services to their consumers.

Some vendors are azure (Azure stack), AWS and so on.

  • Customers are using one single application, but shared data centre.
  • Providing a full infrastructure for an organization on the cloud.
  • Taking public cloud services and providing them on-prem services.

Cloud computing is changing the IT world. For more information, you can refer to google.
